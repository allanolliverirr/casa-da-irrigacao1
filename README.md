# Casa da Irrigação

Este é um projeto de exemplo construído com **Next.js** e **Tailwind CSS** para uma loja fictícia chamada Casa da Irrigação.

O site inclui uma página inicial com seções de autoridade, categorias, produtos em destaque e um formulário de contato via WhatsApp. Também há páginas de catálogo, categorias específicas, páginas de produto, carrinho de compras, sitemap e robots para SEO.

## Como rodar o projeto

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Rode o ambiente de desenvolvimento:
   ```bash
   npm run dev
   ```
3. Acesse `http://localhost:3000` no navegador.

Para construir para produção:

```bash
npm run build
npm start
```
